✅ ¿Qué es este Sistema?

El Sistema de Alertas Tempranas permite a docentes y personal educativo registrar situaciones relevantes del estudiantado que puedan requerir atención por parte del Departamento de Orientación y Equipo para la permanencia institucional. Está diseñado para ser simple, rápido y accesible desde cualquier dispositivo.

👤 ¿Quiénes pueden usarlo?

El sistema está disponible para:

  • Docentes del Liceo de Bebedero
  • Personal administrativo autorizado

Cada registro se guarda de forma segura en una base de datos administrada por el equipo de Orientación.

📝 ¿Cómo registrar una alerta?

  1. Ingrese a la sección "Alertas" desde el menú inferior o lateral.
  2. Seleccione la sección y el nombre del estudiante.
  3. Indique la acción observada y la dimensión correspondiente.
  4. Seleccione un motivo o justificación.
  5. Agregue una observación adicional (opcional).
  6. Ingrese su nombre en el campo "Profesor que reporta".
  7. Presione Enviar para registrar la alerta.

Recibirá un mensaje de confirmación si todo fue exitoso.

🔍 ¿Cómo ver los reportes?

  1. Presione el botón "Reportes" en el menú inferior o lateral.
  2. Ingrese la clave de acceso autorizada.
  3. Una vez validada, podrá ver todos los registros ordenados por fecha.
  4. Puede aplicar filtros por:
    • Fechas: Seleccione una fecha de inicio y fin para ver solo los registros dentro de ese rango.
    • Sección: Use el botón “➕ Filtros Avanzados” para seleccionar una sección específica.
    • Estudiante: También desde los filtros avanzados, puede seleccionar a un estudiante específico.
    • Combinaciones: Puede usar uno o varios filtros al mismo tiempo (ejemplo: sección + estudiante + fecha).
  5. Presione el botón Filtrar para aplicar los criterios.

ℹ️ Recomendaciones

  • Use el sistema de forma responsable y ética.
  • No comparta la clave de acceso a Reportes.
  • Evite registros repetidos o incompletos.
  • Si nota errores o necesita ajustes, contacte al desarrollador.
